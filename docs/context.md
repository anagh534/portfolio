# Project Context: Portfolio

## Overview
This is a high-performance portfolio and service-offering website built using the Astro framework and React. The purpose of this project is to showcase web development and software engineering capabilities while serving as a lead-generation tool for various specialized services (MERN stack, Flutter, WordPress, e-commerce, etc.) across multiple locations in Kerala.

## Tech Stack
- **Framework**: Astro (SSG/hybrid)
- **UI Library**: React (for interactive components)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React

## Core Commands
- `npm run dev`: Start local development server
- `npm run build`: Build for production and generate dynamic sitemap
- `npm run preview`: Preview production build locally
- `npm run lint`: Check syntax using ESLint
- `npm run check`: Astro-specific type checking
